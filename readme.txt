The Elementor Addon For Posts for Elementor Page Builder helps you to create an Elementor List of Posts of any type with three types of layouts: list, grid and carousell

== Description ==
Install WPHD Posts Addon using WordPress plugin installer or by uploading its files into the WordPress plugins directory.
Activate the WPHD Posts Addon through WordPress Plugins menu.
Click the Plugins => Add New
Click Upload Plugin and then Choose the wphd-posts-addon.zip from your computer then click on the Install Now button.
After activating, it doesn't work without Elementor Page Builder plugin. Firstly, you should activate Elementor Page Builder.
To activate newly installed plugin navigate the Plugins => Installed Plugins.
Find the WPHD Posts Addon plugin from the list and press/click Activate.
Now the plugin has been installed and activated on your website.
In the content tab, you can find the following options: layout (grid, list, and carousel), post type (you can choose any post type you want), 
the number of posts to show, and how to order them.There are a lot of options in every single section. If you choose the carousel as a layout, you will see the Carousel section in the style tab. You can change almost everything in the carousel: show or hide arrows or dots, or even both of them, and change the speed. If you select to show arrows you can change the icon, choose some from the Elementor Icon Library, or even add your SVG icon. You can change the width and height of the icon and its container, define to container the background color, border, border-radius etc.
